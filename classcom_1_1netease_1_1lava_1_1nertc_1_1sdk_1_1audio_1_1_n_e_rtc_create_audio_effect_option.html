◆ loopCount

int com.netease.lava.nertc.sdk.audio.NERtcCreateAudioEffectOption.loopCount

音效循环播放的次数：

  • 1：（默认）播放音效一次。
  • ≤ 0：无限循环播放音效，直至调用 stopEffect 或 stopAllEffects 后停止。

◆ path

String com.netease.lava.nertc.sdk.audio.NERtcCreateAudioEffectOption.path

待播放的音效文件路径，支持本地 SD 卡中的绝对路径或 URL 地址。

  • 需精确到文件名及后缀，例如 “/sdcard/test.mp3”。
  • 支持的音效文件类型包括 MP3、M4A、AAC、3GP、WMA 和 WAV 格式。

◆ playbackEnabled

boolean com.netease.lava.nertc.sdk.audio.NERtcCreateAudioEffectOption.playbackEnabled = true

是否本地播放该音效。默认为 true，即本地用户可以听到该音效。

◆ playbackVolume

int com.netease.lava.nertc.sdk.audio.NERtcCreateAudioEffectOption.playbackVolume = 100

音效文件的播放音量，取值范围为 0 ~ 200。默认为 100，表示使用文件的原始音量。

注解
若您在通话中途修改了音量设置，则当前通话中再次调用时默认沿用此设置。
